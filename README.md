# S2DUI
A simple UI library inspired by dear ImGUI running on [my OpenGL rendering library](https://github.com/oliwerdahlin/S2DGL)

#Features
+ Menus
+ Submenus
+ Buttons
+ Checkboxes
+ Sliders
+ Text Input Boxes
